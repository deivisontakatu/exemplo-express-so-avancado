# 🖥️ SO Dashboard

Projeto desenvolvido com **Node.js + Express** para exibir informações do sistema operacional em um dashboard web.

---

# 🚀 Tecnologias

* Node.js
* Express.js
* HTML
* CSS

---

# 📊 Informações exibidas

* Sistema operacional
* Hostname
* Arquitetura
* Usuário atual
* Memória RAM
* CPU
* Rede
* Arquivos do projeto
* Uptime
* Ambiente de execução

---

# 📁 Estrutura

```bash
index.js
package.json
README.md
```

---

# ▶️ Como executar

## Instalar dependências

```bash
npm install
```

## Iniciar projeto

```bash
npm start
```

## Acessar no navegador

```text
http://localhost:3000
```
